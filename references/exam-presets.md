# Exam Presets

## Postgraduate Entrance Exam

Common subjects:

- Math: course slices, example problems, chapter exercises, mistake rework, timed mixed sets, past exams.
- English: vocabulary, reading, long sentences, translation, writing templates, past exams.
- Politics: framework, multiple-choice, confusion points, recitation, current affairs when relevant.
- Major subject: course/textbook chapters, concept maps, recitation outlines, past exam themes, written answers.

Stages:

- Foundation: course + examples + light chapter practice.
- Strengthening: question-bank training + method summaries + mistake rework.
- Past exam: timed past papers, theme analysis, weak-point repair.
- Sprint: recitation, mock exams, formula/framework recall, high-yield mistakes.

Default ratios:

- Foundation: course 40%, exercises 45%, review 15%.
- Strengthening: course 20%, exercises 60%, review 20%.
- Past exam/sprint: course 10%, exams/exercises 70%, review 20%.

## Certification Exams

Use syllabus modules as the main planning unit. Prefer:

- Module checklist
- Question-bank progress
- Wrong-answer themes
- Timed mock cycles
- Final memory sheets

## Language Learning

Use daily rolling loops:

- Vocabulary
- Input: reading/listening
- Analysis: sentence, grammar, pronunciation, or phrase extraction
- Output: speaking/writing
- Review: spaced repetition and weak items

## Daily Skill Learning

Use project or exercise outputs:

- Learn one concept
- Apply it in a small task
- Save notes or examples
- Review or refactor after feedback

Avoid passive-only tasks when the goal is skill acquisition.
